# MediFlow 🏥

A production-grade Healthtech platform built to demonstrate
Cloud & DevOps engineering — CI/CD, IaC, containerization, and AWS deployment.

## Tech Stack
- **Frontend** — React
- **Backend** — NestJS + PostgreSQL
- **Cloud** — AWS (EC2, RDS, S3, ALB, IAM, ACM, Route 53)
- **DevOps** — Docker, GitHub Actions, Terraform

## Local Setup
1. Clone the repo
2. Copy `.env.example` → `.env` in both `frontend/` and `backend/`
3. Fill in values (never commit `.env`)
4. Run `docker-compose up`
